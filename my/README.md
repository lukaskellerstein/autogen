# Poetry

`poetry shell`
`poetry install`

1.  select in VSCode environment (bottom bar)
    or
    Ctrl+Shift+P => Python select interpreter

2.  Enter interpreter path
    paste path that is visible in the terminal after command `poetry shell`

FINALLY
F5
